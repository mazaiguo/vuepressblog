# 路由指南

1. 安装指令
   ```
    pnpm install vue-router
   ```

   

2. 搭建页面`xxx.vue`

   

3. 配置`routes`
   ```
   {
       path:'/login',
       component:()=> import('@/views/login/index.vue'),
       name:'login'
   }
   ```

   

4. 在`App.vue`中使用`router-view`

   ```
   <template>
     <div>
       <router-view></router-view>
     </div>
   </template>
   ```

   

5. 配置完路由后，需要在`mian.ts`中注册
   ```
   import router from './views/router'
   app.use(router)
   ```
